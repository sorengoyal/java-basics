### To Run

#### Using  CMD Line

```
javac src/main/java/org/example/Main.java
java -cp src/main/java org.example.Main
```

#### Using Gradle

```
./gradlew jar
java -jar build/libs/java-basics-1.0-SNAPSHOT.jar
```
